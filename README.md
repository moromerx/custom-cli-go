### Prerequisites
- [Go](https://golang.org/dl/) 1.18 or higher

### Installation
Clone the repository:
```bash
git clone https://github.com/moromerx/custom-cli-go.git
cd custom-cli-go
```

Build the CLI:
```bash
go build -o custom-cli
```

### Usage
Run the CLI:
```bash
./custom-cli
```
